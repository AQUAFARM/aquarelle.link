== Description ==

Sirius Lite is a clean Material Design inspired WordPress theme. It has a custom frontpage template to showcase special timeless elements and information. It also doubles as a one-page theme if needed. Sirius Lite is versatile since it can be used for large and small websites alike, such as landing pages, blogs, company websites, and personal web pages. Sirius Lite features:

* Ability to create a one-page theme
* Ability to change the background color for each front page section
* Multiple blog feed views 
* Ability to turn sidebar on or off on pages and posts
* Custom front page template that allows for multiple sections such as call to action rows, featured pages with icons, featured pages with images, recent blog posts, and open content
* Full width page template
